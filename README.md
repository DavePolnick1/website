# DavidPolnick.com Main Site + Probate Section

This package includes:
- luxury-style main site
- fully built-out probate section at `/probate/`
- executor guide and attorney page
- compliance footer
- headshot, CPRES seal, eXp logo, and Sway logo

## Important note
The earlier generated images were not reused for the probate section. The probate area now uses cleaner branded layouts and user-supplied assets instead of incorrect generated visuals.

## Open in VS Code
1. Extract the zip
2. Open the folder in VS Code
3. Use Live Server for preview

Upload the contents of this folder into `public_html`
