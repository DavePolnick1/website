Key Change Advantage Seller Landing Page
======================================

Files included
--------------
- index.html            -> standalone seller landing page for davidpolnick.com/sell
- mailer.php            -> PHP mail handler
- thank-you.html        -> success page
- thank-you-error.html  -> error page
- README.txt            -> setup notes

Deployment
----------
1. Create a folder named /sell on your hosting account under davidpolnick.com.
2. Upload all files into that folder.
3. Visit davidpolnick.com/sell/ or davidpolnick.com/sell/index.html

Notes
-----
- This version is HTML with a separate PHP mailer.
- The page is standalone and does not link back to the main site.
- The mailer uses PHP's built-in mail() function.
- If your host disables mail(), replace mailer.php with PHPMailer/SMTP.
